Key Projects

UX Design Course Project – UX Portfolio System

The Problem

The goal of this project was to design a complete UX portfolio experience that clearly communicates skills, experience, and project work in a way that is easy for recruiters and clients to evaluate.

Tools Used

  • Sketching
  • Wireframing
  • Prototyping
  • Usability heuristics evaluation

Discovery Process

The UX process included the following:

  • Identifying what information employers care about most
  • Creating layout sketches for page structure
  • Designing an interface focused on clarity and readability
  • Building a prototype and refining the layout during iteration

User Testing

Users provided feedback that emphasized simplicity, organization, and clean navigation. Adjustments were made to spacing, flow, and clarity of project sections.

Usability Heuristics Applied

  • Consistency and standards
  • Recognition over recall
  • Minimalist & clear design
  • Logical hierarchy and navigation

Autonomous Farm Robot Tractor – UX & Control Interface

The Problem

Farmers need a simple, reliable interface to control and monitor an autonomous farm tractor. The UI must function in outdoor conditions, support gloved hands, and provide real-time feedback for GPS position, motor states, and mode switching.

Tools Used

  • Kivy (GUI development)
  • Python
  • Storyboarding and workflow sketches
  • Usability heuristics
  • Real hardware testing on the tractor platform

Discovery Process

The design process focused on:

  • Interviewing farm operators to understand real-world needs
  • Sketching potential control-panel layouts
  • Designing a clean map-centered interface
  • Ensuring critical controls are accessible and easy to see

User Testing

Through hands-on tractor testing, adjustments were made to button sizes, color contrast, and information density. The result was a safer and more intuitive interface for operators.

Usability Heuristics Applied

  • Visibility of system status
  • Error prevention
  • Flexibility and efficiency of use
  • Match between system and real-world terms
